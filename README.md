# 💰 Family Expense Tracker

A simple and interactive **Streamlit-based web application** to manage family income and expenses efficiently. This project helps users track earnings, monitor spending, and visualize financial data in an easy-to-understand way.

---

## 🚀 Features

* 👨‍👩‍👧‍👦 Add and manage family members
* 💵 Record earnings for each member
* 📊 Track expenses across multiple categories
* 🔄 Automatically merge similar expense categories
* 📈 Visualize expense distribution using charts
* 💰 Calculate total earnings, expenses, and remaining balance
* 🎨 Clean and user-friendly UI

---

## 🧠 How It Works

1. Add family members and their earnings
2. Enter expenses under different categories
3. View financial summary (earnings, expenses, balance)
4. Analyze spending using visual charts

---

## 📦 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/Expense-Tracker.git
cd Expense-Tracker
```

---

### 2️⃣ Install Dependencies

```bash
python -m pip install -r requirements.txt
```

---

### 3️⃣ Run the Application

```bash
python -m streamlit run app.py
```

---

### 🌐 Open in Browser

```
http://localhost:8501
```
